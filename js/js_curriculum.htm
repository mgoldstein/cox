JavaScript 101

 

 

What’s the difference between Java and Javascript?

 

                Client-side: the user’s browser interprets Javascript. Different browsers read Javascript differently. Most personal web page hosts (Earthlink, Geocities) will not allow you to use CGI. But Javascript is built into your web browser, so you can use JS on personal web pages.

 

                For security reasons, JS cannot open or manipulate files on either the client’s machine or the host server. I’ve read that JS can be used for server-side functions, but I’ve never seen it in the real world. There are too many other server-side languages that are superior (PERL, PHP, ASP, JSP, etc)

Concepts & Vocab: Objects (properties & methods), functions (operators), variables (arrays), events.

 

                The browser is a stage. The actors and the sets are objects. The lines spoken and actions performed are functions and methods. The actors’ cues are events.

 

Variables: Store information

                Should start with a letter

                Must be one word

                Avoid symbols (underscore ok)

                Beware reserved words

                Case-sensitive

                Arrays

 

Functions: Perform a task and return a value. Allows greater control over when code is executed.

                Beware reserved words

                Case-sensitive               

                Passing variables into functions.

In calls to functions, the parentheses can be left empty or used to pass a value to the function.

In the function definition, the parentheses are used to assign variable names to the values passed from the calls.

                If statements:

                                Operators (>, <, ==, !=, >=, <=, &&, ||, ++, --, *, /)

 

Objects:  Parts of the browser.

                Objects are parts of the browser that you can access and modify. Objects may have sub-elements, called properties. Objects are manipulated through methods and functions.

                window

Properties: self, status, parent

Methods: alert, prompt, confirm, close, open

                date (no properties, many methods)

                document

Properties: location, title, forms, cookie, bgcolor)

                                Methods: write, open, close

 

Events: Call out to functions and methods.

                OnLoad, onClick, onMouseOver, onMouseOut
